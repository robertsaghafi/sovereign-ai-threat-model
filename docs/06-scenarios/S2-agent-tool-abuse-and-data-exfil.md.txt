# Scenario 2: Agent Tool Abuse and Data Exfiltration

## Overview

An enterprise deploys an agentic AI assistant capable of retrieving documents and interacting with internal APIs.

The agent is granted access to document search tools and analytics platforms.

A malicious prompt causes the agent to retrieve and expose sensitive information.

## Threat Path

1. User issues prompt requesting analysis
2. Prompt includes hidden instructions requesting additional data
3. Agent retrieves documents from internal knowledge base
4. Agent includes sensitive information in response

## Risks

- Excessive data retrieval
- Unauthorized document access
- Data exfiltration through generated output

## Business Impact

- Disclosure of confidential internal documents
- Exposure of regulated information
- Loss of intellectual property

## Mitigation Strategies

- Enforce role-based access control for agents
- Limit agent tool permissions
- Apply data classification filtering before output generation
- Monitor agent activity logs

## Governance Alignment

Controls should align with:

- Zero Trust access principles
- Data governance policies
- AI risk monitoring processes
